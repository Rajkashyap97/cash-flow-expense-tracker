# Development Note

This project was developed as part of Sprint 02: Cash-Flow (Salary & Expense Tracker).

The primary objective of this sprint was to strengthen core JavaScript concepts including DOM Manipulation, Event Handling, State Management, Data Persistence, and API Integration using Vanilla JavaScript.

I independently implemented the application's core functionality, including:

* Salary and Expense Management
* Dynamic DOM Updates
* Real-Time Balance Calculation
* Input Validation
* LocalStorage Data Persistence
* Expense Delete Operations
* Chart.js Data Visualization
* Currency Conversion Integration
* Threshold Warning Alerts
* PDF Report Generation

During development, I utilized AI tools, official documentation, and online developer resources for research, debugging, optimization, and understanding implementation approaches. Similar to industry-standard development practices, these resources were used as references and learning aids.

The overall application structure, business logic, feature integration, testing, debugging, customization, and final implementation were completed and validated by me.

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript (ES6)
* Chart.js
* jsPDF
* LocalStorage API
* Frankfurter Exchange Rate API

## Learning Outcomes

* DOM Manipulation
* Event Listeners
* State Management
* Data Persistence
* API Consumption
* Dynamic Rendering
* Client-Side Report Generation

This project reflects my practical understanding of frontend engineering fundamentals and JavaScript application development.
